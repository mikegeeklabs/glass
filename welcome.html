GLASS Version 0.1 2014-12-21, Winter Solstice Edition

The basic applation framework and key pieces all work well enough to use in other projects that I need to start work on.

GLASS Goals

  1. A reasonably secure framework for basic PHP/MySQL/MariaDB applications.
  2. Support for the PHP Native MySQLI connectors
  3. Useful for both web and CLI usage
  4. Multi-language/timezone support
  5. Table Data Editing and Reporting
  6. Clear Easy To Read/Maintain Code
  7. Minimal CSS and no jQuery required.
  8. Using HTML5 for Datepickers and other elements

Instructions:

  1. Create database (MySQL/MariaDB) and load 'database.sql'
  2. Edit settings.inc (note; configure webserver to not deliver .inc files)
  3. Make sure HTTPS is configured for this site. "basic auth" is only secure when using HTTPS, which you should be using anyway. StartSSL.com is worth exploring.
  4. Use 'glass.php' - default login is: admin /1234 (yes, you need to change that password, edit the users table. 'passwd' field is bcrypt hashed)
